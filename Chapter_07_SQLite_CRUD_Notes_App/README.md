# sqlite_app

A Chapter 7 hands-on practice app for SQLite CRUD in Flutter.

## Files for the practice

- `lib/main.dart` starts the app.
- `lib/homepage.dart` shows the notes screen.
- `lib/db_service.dart` handles SQLite CRUD.

Run `flutter pub get` before running the app so Flutter downloads `sqflite`.
