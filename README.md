# Mobile Application System Design Class Work

This repository contains the in-class practice projects for Mobile Application System Design.

`Group_Project` is intentionally not included in the class-work list below.

## Submission Contents

| Lecture | Folder | Topic |

| Chapter 03 - Part 2 | `Chapter_03_Part_2_Restaurant_App` | Restaurant app UI, images, icons, TextField, ListView |
| Chapter 04 | `Chapter_04_Contact_List_ListView` | Contact list app using ListView |
| Chapter 05 | `Chapter_05_Food_Category_GridView` | Food category app using GridView |
